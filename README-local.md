# HealSync local run

This project currently runs as one local Next.js app. The local account, profile,
and notes are stored in the browser's `localStorage`.

## Start

From the project root:

```bat
start-local.bat
```

Or from `frontend-new`:

```bat
npm.cmd run local
```

Open:

```text
http://localhost:3000
```

The `backend` folder is kept in the repository, but it is not required for the
current local-only workflow.
